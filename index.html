Q: When would you use ‘=’ vs ‘==’ vs ‘.equals’? Explain what each of these does and in which cases you should use them.

2) How would you use ‘=‘, ‘==‘, and .equals? Well, ‘=‘ is used to assign a value to a variable. If you were to use ‘==‘ inste

ad of ‘=‘ it wouldn’t work because it would be comparing the two, not assigning a value to the variable. An example of how

to use ‘=‘ to assign a value to a variable is as follows: myAge = 30; Now, whenever I would invoke myAge it would be tied to

the value 30. For example, if I were to use System.out.println(“I’m “ + myAge + “.”); —— It would display “I’m 30.” You would

use ‘==‘ in order to compare integers, chars, and booleans, etc. For example: 4 == 5 The above integers would be compared and

the statement would be determined to be false. It also would not assign the value 5 to 4 because in this case 4 is not being

used as a variable as the previous example. Finally, you would use ‘.equals’ to verify to see if two Strings have the same val

ue. If you were to use ‘==‘ then you would not be able to verify if they had the same value properly as it would return

false. For example, it’s true that:

String slacker = “Fred”; String slacker has the same value as String worker. String worker = “Fred”; But if you were to do slacker.equals(worker);

it would be a false statement.
